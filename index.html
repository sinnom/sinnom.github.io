Sinnom: Advent of Code 2022

2022-12-04

This is my first year doing Advent of Code. You can view my code for each day on my github.

I am working in Rust and using iterator traits from std and itertools, though I might use other tools as the event progresses.

All of my solutions except day 2 use exclusively iterators and the itertools crate. Day 2 primarily uses rust enums, impl blocks, and the From trait.
